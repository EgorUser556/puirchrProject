# PuirchrProject

A web application for self‑assessment of HR skills such as delegation, recruitment, onboarding, motivation, compensation, and team building. Each test uses a 1–5 Likert scale and shows a result page with an aggregated score and interpretation.

## Features

- Collection of short HR tests grouped by topics.
- Test selection page with clear navigation between topics and individual tests.
- Step‑by‑step answering on a 1–5 scale for each question.
- Result page with calculated score and basic recommendations.
- Shared layout with header, footer, and client‑side routing.

## Tech Stack

- React (functional components, hooks)
- React Router
- Create React App / custom webpack dev server (`npm start`)
- HTML5, CSS3 (responsive layout)

## Getting Started

1. Clone the repository: git clone https://github.com/EgorUser556/puirchrProject
2. Install dependencies: npm install
3. Start the development server: npm start
4. Open the app in the browser: Usually available at `http://localhost:3000`.
